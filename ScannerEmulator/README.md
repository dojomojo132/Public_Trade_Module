# PTM Scanner Emulator

Эмулятор USB сканера штрихкодов для 1С PTM.

## Возможности

- ✅ Полная имитация клавиатурного ввода (как настоящий USB сканер)
- ✅ Без префиксов и специальных маркеров
- ✅ Графический интерфейс с быстрыми кнопками
- ✅ История сканирований
- ✅ Хоткей Ctrl+Shift+S для быстрого доступа

## Установка

### Вариант 1: PowerShell (простой)

```powershell
# Разовое сканирование
.\ScanSimple.ps1 -Barcode "4820000123456"

# Интерактивный режим
.\ScanSimple.ps1
```

### Вариант 2: Node.js + Electron (с GUI)

```bash
# Установка зависимостей
npm install

# Запуск
npm start

# Сборка .exe
npm run build:win
```

## Использование

1. Запустите эмулятор
2. Откройте форму РМК в 1С
3. Установите фокус в поле поиска товара
4. В эмуляторе введите штрихкод и нажмите Enter
5. Или используйте быстрые кнопки

## Системные требования

**PowerShell:**
- Windows 7+
- PowerShell 5.0+

**Electron:**
- Windows 10/11
- Node.js 18+
- 100 MB свободного места

## Отличия от настоящего USB сканера

| Параметр | USB сканер | Эмулятор |
|----------|-----------|----------|
| Ввод символов | ⚡ Мгновенно | ⏱️ ~10ms/символ |
| Prefix/Suffix | Настраивается | Только Enter |
| Работа в фоне | ❌ Требует фокус | ❌ Требует фокус |

## Тестовые штрихкоды

```
4820000123456 - Тестовый товар 1
4820000654321 - Тестовый товар 2
5000000000001 - Молоко
5000000000002 - Хлеб
```
