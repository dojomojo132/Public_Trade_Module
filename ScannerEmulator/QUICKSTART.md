# 🔷 PTM Scanner Emulator - Краткая справка

## Проблема
USB сканеры не имеют настройки префиксов → эмулятор должен работать как настоящая клавиатура.

## Решение
Эмулятор имитирует **клавиатурный ввод** через SendKeys/robotjs - полностью совместимо с USB сканерами.

---

## ⚡ Быстрый старт

### 1. PowerShell (рекомендуется)
```powershell
START.bat
# → [1] PowerShell - Простой
```

### 2. Тестовое сканирование
```powershell
START.bat
# → [3] Тестовое сканирование
```

### 3. Интерактивный режим
```powershell
.\ScanSimple.ps1
# Введите штрихкод: 4820000123456
```

### 4. Electron GUI (требует npm install)
```bash
npm install
npm start
```

## 🔄 Переключение на USB сканер

**Требуется:** 0 изменений в коде! ✅

1. Подключить USB сканер
2. Открыть РМК
3. Сканировать товар
4. **Работает!**

Текущий код РМК с пинг-понг полями уже готов для USB сканеров.
