# Справочник элементов управляемых форм 1C 8.3.24

## Основные элементы

### Поле ввода (InputField)
```xml
<Item xsi:type="InputField">
    <Name>Наименование</Name>
    <DataPath>Объект.Наименование</DataPath>
    <Width>30</Width>
    <AutoMaxWidth>true</AutoMaxWidth>
</Item>
```

**Обработчики:**
- `ПриИзменении` — после ввода значения
- `НачалоВыбора` — открытие формы выбора
- `Очистка` — очистка значения

### Таблица формы (FormTable)
```xml
<Item xsi:type="Table">
    <Name>Товары</Name>
    <DataPath>Объект.Товары</DataPath>
    <CommandBarLocation>Top</CommandBarLocation>
    <ChangeRowSet>true</ChangeRowSet>
    <ChangeRowOrder>true</ChangeRowOrder>
</Item>
```

**Колонки таблицы:**
```xml
<Items>
    <Item xsi:type="InputField">
        <Name>ТоварыНоменклатура</Name>
        <DataPath>Объект.Товары.Номенклатура</DataPath>
    </Item>
    <Item xsi:type="InputField">
        <Name>ТоварыКоличество</Name>
        <DataPath>Объект.Товары.Количество</DataPath>
        <Format>ЧДЦ=3</Format>
    </Item>
</Items>
```

### Кнопка (Button)
```xml
<Item xsi:type="Button">
    <Name>КнопкаОплатить</Name>
    <Title>Оплатить (F9)</Title>
    <CommandName>Форма.Команда.Оплатить</CommandName>
    <Shortcut>F9</Shortcut>
</Item>
```

### Группа элементов
```xml
<Item xsi:type="UsualGroup">
    <Name>ГруппаШапка</Name>
    <Group>HorizontalIfPossible</Group>
    <Representation>None</Representation>
</Item>
```

## Программное управление элементами

### Видимость/Доступность
```bsl
&НаКлиенте
Процедура УстановитьВидимостьЭлементов()
    Элементы.КнопкаОплатить.Видимость = СуммаДокумента > 0;
    Элементы.Товары.ТолькоПросмотр = Статус = Перечисления.СтатусыЧекаККМ.Пробит;
КонецПроцедуры
```

### Динамическое создание элементов
```bsl
&НаСервере
Процедура СоздатьКнопкиТоваров(Группа)
    Для Каждого Товар Из БыстрыеТовары Цикл
        НоваяКнопка = Элементы.Добавить("Кнопка_" + Товар.Код, Тип("КнопкаФормы"), Группа);
        НоваяКнопка.Заголовок = Товар.Наименование;
        НоваяКнопка.ИмяКоманды = "ДобавитьБыстрыйТовар";
        НоваяКнопка.Ширина = 15;
    КонецЦикла;
КонецПроцедуры
```

## Типовые обработчики РМК

### Обработка сканирования
```bsl
&НаКлиенте
Процедура ПолеПоискаТовараПриИзменении(Элемент)
    Штрихкод = СокрЛП(ПолеПоискаТовара);
    Если ПустаяСтрока(Штрихкод) Тогда
        Возврат;
    КонецЕсли;
    
    ДобавитьТоварПоШтрихкоду(Штрихкод);
    ПолеПоискаТовара = "";
КонецПроцедуры

&НаКлиенте
Асинх Процедура ДобавитьТоварПоШтрихкоду(Штрихкод)
    РезультатПоиска = НайтиТоварНаСервере(Штрихкод);
    
    Если РезультатПоиска.Количество() = 0 Тогда
        ПоказатьПредупреждение(, "Товар не найден: " + Штрихкод);
    ИначеЕсли РезультатПоиска.Количество() = 1 Тогда
        ДобавитьВКорзину(РезультатПоиска[0]);
    Иначе
        // Несколько товаров — показать выбор
        Оповещение = Новый ОписаниеОповещения("ПослеВыбораТовара", ЭтотОбъект);
        ПоказатьВыборИзСписка(Оповещение, РезультатПоиска, Элементы.ПолеПоискаТовара);
    КонецЕсли;
КонецПроцедуры
```
