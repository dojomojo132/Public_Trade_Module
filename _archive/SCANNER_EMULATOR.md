# 🔷 PTM Scanner Emulator

## Эмулятор USB сканера штрихкодов для тестирования

### Расположение
`ScannerEmulator/`

### Описание
Программное решение для имитации работы физического USB сканера штрихкодов.
Использует клавиатурный ввод (SendKeys) - полностью совместимо с реальными сканерами.

### Быстрый запуск
```powershell
cd ScannerEmulator
START.bat
```

### Переход на USB сканер
**Не требует изменений в коде!** Просто подключите сканер и он будет работать.
